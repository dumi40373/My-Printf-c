# Printf – Reproduction en C

## Description
Reproduction de la fonction printf en C.

## Compétences travaillées
- Parsing
- Gestion mémoire
- Gestion des flags
- Respect des normes EPITECH

## Compilation
make

## Exemple d'utilisation
./my_printf "Hello %s" "World"
